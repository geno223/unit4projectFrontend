### React Router Routes:
- Home Page: "/"
  - Display items for sale and link to Mens, Womens & Kidz apparel.

- Item Detail Page: "/items/:itemId"
  - Display detailed information about a specific item.

- Create Item Page: "/create-item"
  - Allow users to create new items.
